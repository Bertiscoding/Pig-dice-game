Instructions:

The dice rolls and displays a random number between 1-6. Each time the dice result gets added to the current score.

The current player can roll or hold the dice. If they choose to hold it, the current score gets added to the global score and it's the next players turn.

If a 1 is rolled, the current score resets to 0 again and their turn is over. The program switches to the next player.

The first player who reaches 30 wins!
